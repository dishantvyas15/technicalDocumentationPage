Introduction

This page describes the usage of five of the most important topics of the C++ language, namely, for-loop, while-loop, do-while loop, break command, and continue command.

The For-Loop

For-loop is used when statement(s) is/are required to be iterated an exact number of times.
Here is an example code:

for(int i=1 ; i<=10 ; i++) {
    cout << i;
}

  • It can be used without curly braces ( { } ) if it contains only one statement.
  • It can be used with curly braces ( { } ) if it contains at least one statement.
The While-Loop

While-loop is used when the number of required iterations is unknown and is based on a certain condition.
Here is an example code:

int i=10;
while(i*i<1000) {
    cout << i;
    i += 2;
}

  • The statements within the loop get executed as long as the specified condition remains true.
  • Here, the condition is checked prior to the execution of the loop statements.
The Do-While Loop

While-loop is used when the number of required iterations is unknown and is based on a certain condition.
Here is an example code:

int i=10;
do {
    cout << i;
    i += 2;
}while(i*i<1000)

  • The statements within the loop get executed as long as the specified condition remains true.
  • Here, the loop statements are executed prior to the condition check.
The Break Command

The break statement gives us the legacy to "break out" of a loop as soon as the execution reaches a certain point in the loop (usually, a condition). Here is an example code:

for(int i=1 ; i ; i++) {
    cout << i;
    if(i==25) {
        break;
    }
}

  • This command lets the execution break out of only a single loop.
  • It is often used when there is no governing loop condition.
The Continue Command

The continue statement gives us the legacy to return the execution in a loop to the beginning of the loop. Here is an example code:

for(int i=1 ; i<=10 ; i++) {
    if(i%2==0) {
        continue;
    }
    cout << i;
}

  • This command shifts the control to the beginning to the loop (starting the next iteration).
  • It is often used when the execution of the loop statement(s) for some of the values of the loop variable is to be skipped.
